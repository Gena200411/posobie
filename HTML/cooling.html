Что такое системы охлаждения?

❄️ "Терморегуляция" ПК

Система охлаждения (Cooling) — это комплекс устройств для отвода тепла от нагревающихся компонентов компьютера: процессора, видеокарты, чипсета, памяти, накопителей.

🌡️ Почему нужно охлаждение?

При перегреве:

  • Троттлинг — принудительное снижение частоты
  • Нестабильность — вылеты программ, BSOD
  • Снижение ресурса — деградация чипов
  • Выход из строя — критический перегрев

📊 Основные задачи

  • Отвод тепла — от CPU, GPU, VRM, чипсета, RAM
  • Отвод горячего воздуха — из корпуса наружу
  • Шумопонижение — эффективное охлаждение на низких оборотах
  • Стабильность — обеспечение работы ПК под нагрузкой

🌡️ TDP рассеиваемая мощность (Вт)

Сколько тепла может отвести система охлаждения. Кулер должен иметь TDP ≥ TDP процессора.

×

🔇 Уровень шума (дБА)

25-30 дБА — фоновый шум, 40+ дБА — слышно.

×

🌀 Скорость вращения (RPM)

Регулировка PWM (4-pin) лучше DC (3-pin).

×

💨 Воздушный поток (CFM)

Для корпусных вентиляторов важнее airflow.

×

⛲ Статическое давление

Для кулеров/радиаторов важнее статическое давление.

×

📏 Габариты и совместимость

Высота кулера должна влезать в корпус (160+ мм для ATX).

×

🔬 Как производят

  1. Литьё основания радиатора, тепловых трубок
  2. Пайка трубок к основанию
  3. Изготовление вентилятора
  4. Сборка СЖО (герметизация, заправка жидкостью)
  5. Контроль качества, тесты на утечку

💨 Воздушное охлаждение

Радиатор + тепловые трубки + вентиляторы. Самый распространённый тип.

💧 СЖО (All-in-One)

Готовая система: водоблок, помпа, трубки, радиатор.

🔧 Кастомная вода

Сборная система для энтузиастов.

×

🔇 Пассивное охлаждение

Без вентиляторов (только радиатор). Тишина, но ограниченная производительность.

×

🧊 Фазовый переход (Peltier)

Может работать ниже комнатной температуры, дорого, высокое энергопотребление.

×

💨 Жидкий азот (LN2)

Экстремальный разгон (-196°C). Только для бенчмарков и рекордов.

🛢️ Подшипники

  • Втулка (Sleeve) — дешёвые, малый ресурс
  • Гидродинамический (FDB) — тихие, долгие
  • Шариковый (Ball) — шумные, вечные
×

🌈 RGB (12V) vs ARGB (5V)

RGB — 1 цвет на все светодиоды. ARGB — индивидуальная подсветка каждого светодиода.

×

🎛️ PWM (4-pin)

Плавное регулирование в широком диапазоне через ШИМ-сигнал.

×

📊 Airflow vs Static Pressure

Airflow — для продува корпуса. Static Pressure — для радиаторов, плотных фильтров.

💡 Как выбрать систему охлаждения?

📌 По назначению

  • Офисный ПК: Боксовый кулер (в комплекте), Arctic Alpine, Deepcool GammaXX
  • Игровой ПК (бюджет): Башенный кулер 120 мм (Cooler Master Hyper 212, Deepcool AG400)
  • Игровой ПК (премиум): Башня 120-140 мм / AIO 240-360 мм
  • Компактный/малошумный HTPC: Пассивное / Noctua NH-L9 / be quiet! Pure Rock LP
  • Экстремальный разгон: Noctua NH-D15 / Deepcool AK620 / AIO 360+ / Кастомная вода / LN2
  • Для слабых CPU: боксовый кулер (TDP 65 Вт)
  • Для i5 / Ryzen 5: башня 120 мм
  • Для i7 / Ryzen 7: башня 140 мм / AIO 240-280 мм
  • Для i9 / Ryzen 9: топовая башня / AIO 360 мм / кастомная вода

⚠️ Важные моменты при выборе

  • Габариты: высота кулера (замерьте корпус)
  • Совместимость с оперативной памятью (высокие радиаторы RAM могут мешать башне)
  • TDP кулера должен быть заявлен на тепловыделение процессора
  • Крепления для сокетов Intel/AMD (особенно для новых сокетов LGA1851, AM5)
