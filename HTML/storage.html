Что такое накопитель данных?

💾 "Долговременная память" ПК

Накопитель данных (Storage) — устройство для долговременного хранения информации. В отличие от оперативной памяти, данные сохраняются после выключения питания.

⚡ Принципы работы

HDD (Hard Disk Drive): магнитные пластины + считывающие головки (механический).
SSD (Solid‑State Drive): ячейки флэш-памяти NAND.

SSD компактнее, быстрее, надёжнее (ударов), но дороже за 1 ГБ.

📊 Основные функции

  • Долговременное хранение данных — ОС, программы, документы, медиа
  • Быстрый доступ — чтение и запись данных по запросу системы
  • Загрузка ОС и ПО — от скорости накопителя зависит время запуска
  • Резервное копирование — хранение бэкапов

📖 Скорость последовательного чтения

Важна для загрузки ОС/игр. SATA SSD: 560 МБ/с, PCIe 3.0: 3500 МБ/с, PCIe 4.0: 5000-7400 МБ/с, PCIe 5.0: 10000-15000 МБ/с.

×

✍️ Скорость последовательной записи

Важна при копировании больших файлов. Аналогичные чтению значения, но может быть ниже.

×

⚡ IOPS

Количество операций ввода-вывода в секунду (чтение/запись мелких блоков). Критично для серверной работы, БД, виртуализации. HDD ~ 100-200, SATA SSD ~ 100k, NVMe ~ 500k-1,5M+.

×

🛡️ Ресурс TBW (TeraBytes Written)

Объём данных, который можно записать до выхода из строя. QLC: ~ 200-400 TBW, TLC: 600-1200 TBW, MLC (>2000 TBW).

×

💾 DRAM / SLC кэш

DRAM-кэш ускоряет операции с таблицей флеш‑преобразований. SLC-кэш (pSLC) — быстрая область на диске для ускорения записи.

×

💽 Ёмкость

HDD: 1-22 ТБ. SSD: 120 ГБ — 8 ТБ (потребительские), 15-60 ТБ (серверные).

×

🔬 Как производят накопители

HDD: создание магнитных пластин, головок, гермоблока, электроники.
SSD: производство чипов NAND (слои 3D NAND), контроллера, монтаж на печатную плату.

💿 HDD (Hard Disk Drive)

  • Механический, накопитель на магнитных пластинах
  • + дешёвый, большие объёмы (до 22+ ТБ)
  • - медленный (150-250 МБ/с), шумный, боится ударов
  • Для хранения больших данных, бэкапов, NAS
×

💿 SATA SSD

  • 2.5-дюймовый, интерфейс SATA 6 Гбит/с
  • + 500-560 МБ/с, бесшумно, не боится ударов, недорого
  • - медленнее NVMe
  • Для апгрейда старых ПК, недорогие хранилища
×

⚡ NVMe SSD

  • Форм-фактор M.2 (реже U.2, AIC). Интерфейс PCIe x2/x4
  • + Скорость 3000-15000 МБ/с, низкая задержка, прямой доступ к CPU
  • - Дороже SATA, греется
  • Для игр, профессиональных задач, системы
×

💿 SSHD (Solid State Hybrid Drive)

Гибрид HDD + маленький SSD смарт-кэш для ускорения частых данных. Компромисс, практически устарел.

×

🔧 Технологии NAND памяти

  • SLC (1 бит/яч) — дорого, отказоустойчиво
  • MLC (2 бита) — дорого, высокий ресурс
  • TLC (3 бита) — современный стандарт
  • QLC (4 бита) — дёшево, ресурс ~200 TBW
  • 3D NAND — вертикальные слои (64–300+) увеличивает плотность
×

📀 Внешние накопители

Портативные HDD (2.5") / SSD с интерфейсом USB 3.x / USB-C / Thunderbolt. Для резервного копирования, переноса данных между ПК.

🔌 SATA III (6 Гбит/с)

Теоретическая скорость ~600 МБ/с. HDD и SATA SSD. Подключаются к разъёмам на плате (обычно 4-8 портов).

×

⚡ PCIe (M.2 / AIC)

M.2: PCIe x2/x4 (NVMe). PCIe 3.0 (3500 МБ/с), 4.0 (до 7500 МБ/с), PCIe 5.0 (до 15000 МБ/с).

×

📀 USB / Thunderbolt

USB 3.2 Gen 2 (10 Гбит/с), USB 3.2 Gen 2x2 (20 Гбит/с) и Thunderbolt 3/4 (40 Гбит/с) — для внешних накопителей.

×

🔧 SAS (Serial Attached SCSI)

Серверный интерфейс: скорость 12/24 Гбит/с, поддержка очередей команд, высокая надёжность (24/7).

×

💡 U.2 (SFF‑8639)

Форм-фактор для высокопроизводительных серверных SSD. Использует PCIe x4, горячая замена.

×

🔌 M.2 Key M (NVMe) / Key B+M (SATA)

Key M — только NVMe, Key B+M — SATA или NVMe. Длина: 2280 (стандарт).

💡 Как выбрать накопитель?

📌 По назначению

  • Система + игры: NVMe SSD (PCIe 3.0 / 4.0) 500 ГБ — 2 ТБ
  • Хранилище данных, бэкапы: HDD 2-8 ТБ (или более)
  • Бюджетная сборка: SATA SSD 240-512 ГБ + HDD 1-2 ТБ
  • Компактный ПК / ноутбук: NVMe SSD M.2 (единственный накопитель)
  • Сервер / NAS: HDD (NAS/Enterprise) с большим объёмом, NVMe SSD (кэш)
  • Для максимальной производительности: NVMe PCIe 4.0 / 5.0
  • Оптимальный вариант для игр: NVMe PCIe 3.0 (3500 МБ/с)
  • Апгрейд старого ПК: SATA SSD
  • Для хранения фото/видео/бэкапов: HDD 4+ ТБ
  • Для NAS/сервера: HDD WD Red / Seagate IronWolf

⚠️ Важные моменты при выборе

  • Наличие DRAM-кэша ускоряет работу и увеличивает ресурс
  • TBW (ресурс записи) важен для активного использования
  • Не переплачивайте за PCIe 5.0 в играх (прирост минимальный)

⚡ PCIe 6.0 / 7.0

PCIe 6.0: скорость до 256 ГБ/с, выход 2025-26. PCIe 7.0 ожидается 2028+. Рост производительности продвинутых SSD.

×

🔌 CXL (Compute Express Link)

Технология объединения памяти и устройств хранения, позволяет использовать SSD как расширение оперативной памяти.

×

🔥 HAMR / MAMR (HDD)

Технологии лазерного (HAMR) или микроволнового (MAMR) разогрева для увеличения плотности записи до 80 ТБ и выше.

×

♻️ Энергоэффективность

SSD с низким энергопотреблением для ЦОД и ноутбуков.

×

📀 Устаревание SATA

Бюджетные NVMe (DRAM-less) дешевеют и вытесняют SATA SSD в ноутбуках и настольных ПК.

×

💾 Гибридная память

MRAM, ReRAM — быстрее NAND, не энергозависимы, потенциал замены DRAM в будущем.
