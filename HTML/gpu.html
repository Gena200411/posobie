Что такое видеокарта?

🎨 "Художник" компьютера

Видеокарта (GPU, Graphics Processing Unit) — это устройство, отвечающее за обработку графики и вывод изображения на монитор. Современные GPU также используются для параллельных вычислений (майнинг, AI, научные расчёты).

⚡ Как работает GPU?

GPU имеет тысячи ядер, оптимизированных для параллельной обработки данных. В отличие от CPU (несколько мощных ядер), GPU использует множество простых ядер для одновременной обработки миллионов пикселей.

Сравнение архитектур CPU и GPU: CPU — несколько мощных ядер, GPU — тысячи простых ядер.

📊 Основные функции GPU

  • Рендеринг 2D/3D графики — обработка игр, интерфейса, видео
  • Видеокодирование/декодирование — NVENC, AMD VCE
  • Параллельные вычисления — CUDA, OpenCL, машинное обучение
  • Ускорение AI и нейросетей — Tensor ядра (NVIDIA), AI Accelerator (AMD)

💾 Объём видеопамяти (VRAM)

Хранит текстуры, кадры, буферы. Влияет на разрешение и качество текстур.

  • 4-6 ГБ — 1080p, старые игры
  • 8-10 ГБ — 1080p-1440p, современные игры
  • 12-16 ГБ — 1440p-4K, новые игры
  • 20-24+ ГБ — 4K, профессиональные задачи
×

🔢 Ядра CUDA / Compute Units

NVIDIA: CUDA ядра; AMD: Compute Units + Stream Processors.

  • Чем больше ядер — тем выше производительность
  • RTX 4090: 16384 CUDA ядер
  • RX 7900 XTX: 6144 Stream процессоров
×

⚡ Частота GPU и памяти

Базовая и Boost частота ядра. Частота памяти в МГц.

  • Boost: до 2.5-3.0 ГГц на современных GPU
  • GDDR6X: до 21 Гбит/с
  • Разгон (OC) увеличивает производительность
×

🔌 Шина памяти (бит)

Пропускная способность между GPU и VRAM.

  • 128 бит — бюджетные карты
  • 192-256 бит — средний сегмент
  • 320-384 бит — топовые карты (RTX 4090 — 384 бит)
×

🌡️ TDP / TGP

Энергопотребление и тепловыделение видеокарты.

  • 75 Вт — без доп. питания
  • 150-250 Вт — средний сегмент (RTX 4070)
  • 300-450+ Вт — топовые модели (RTX 4090)
×

✨ RT и Tensor ядра (NVIDIA)

  • RT ядра — аппаратное ускорение трассировки лучей
  • Tensor ядра — ускорение AI, DLSS
  • AMD — аналоги: Ray Accelerators, AI Accelerators
×

🔬 Как создают видеокарты

  1. Разработка чипа — проектирование архитектуры GPU
  2. Производство чипа — TSMC, Samsung (5-4 нм техпроцесс)
  3. Монтаж на PCB — пайка чипа и компонентов
  4. Установка памяти — монтаж чипов GDDR6/GDDR6X
  5. Система охлаждения — радиаторы, теплотрубки, вентиляторы
  6. Тестирование — проверка стабильности, производительности
  7. Прошивка BIOS/UEFI — задание частот, лимитов

📺 Встроенная графика (iGPU)

  • Интегрирована в процессор (Intel UHD, AMD Radeon Graphics)
  • Общая память с CPU (использует RAM)
  • Для офиса, видео, старых игр
  • Энергоэффективно
×

🎮 Дискретная видеокарта

  • Отдельный GPU с собственной VRAM
  • Высокая производительность для игр и профессиональных задач
  • Требует доп. питания и охлаждения
  • Примеры: RTX 4070, RX 7800 XT
×

💼 Профессиональные GPU

  • Сертифицированные драйверы для CAD/CAM/3D
  • NVIDIA Quadro / RTX A-Series / RTX Ada
  • AMD Radeon Pro / Instinct
  • Большой объём VRAM, ECC память
  • Для научных расчётов, AI, рендеринга

🇷🇺 Отечественные разработки GPU

🔷 Gravitator (GS Nanotech / МЦСТ)

  • Разработка отечественного графического процессора
  • Совместимость с API OpenGL, Vulkan
  • Ориентация на госсектор и импортозамещение

🔶 Микрон (Зеленоград)

  • Разработка графических контроллеров для встраиваемых систем
  • Поддержка отечественных вычислительных систем

🔹 Встроенная графика в Эльбрус и Байкал

  • Процессор Эльбрус-8СВ — встроенный графический контроллер
  • Собственный графический стек на Linux
  • Базовые возможности 2D/3D графики

⚡ Важно: Собственные дискретные видеокарты в РФ находятся на ранней стадии разработки. Планируется выпуск GPU для госсектора в рамках программ импортозамещения.

💡 Как выбрать видеокарту?

📌 По назначению

  • Офис/интернет: Встроенная графика (iGPU)
  • 1080p Gaming: RTX 4060 / RX 7600
  • 1440p Gaming: RTX 4070 / RX 7700 XT / RX 7800 XT
  • 4K Gaming: RTX 4080 / RX 7900 XT / RTX 4090
  • Профессиональные задачи (3D, AI): RTX 4090 / Quadro / Radeon Pro
  • Бюджетный игровой ПК (1080p): RTX 4060 / RX 7600
  • Средний игровой ПК (1440p): RTX 4070 Super / RX 7800 XT
  • Топовый игровой ПК (4K): RTX 4080 Super / RX 7900 XTX
  • Для майнинга: NVIDIA GP104 / AMD Navi — но сейчас не актуально
  • Для AI/ML (бюджет): RTX 3060 12GB / RTX 4060 Ti 16GB
  • Для AI/ML (профессионалам): RTX 4090 / A100 / H100

⚠️ Важные моменты при выборе

  • Соотношение цена/производительность — сравнивайте в бенчмарках
  • Учитывайте блок питания (калькулятор мощности)
  • Проверяйте совместимость с корпусом (длина видеокарты)
  • Для RTX 40xx и RX 7xxx — поддержка PCIe 4.0/5.0 важна
  • Реальные бенчмарки важнее, чем объём VRAM и частоты

🧩 Чиплетная архитектура

AMD уже использует чиплеты (RDNA 3), NVIDIA планирует переход в Blackwell. Это позволит снизить себестоимость и увеличить масштабируемость.

⚡ PCIe 5.0 / 6.0

PCIe 5.0 уже в RTX 40xx (частично), PCIe 6.0 в разработке. Увеличение пропускной способности в 4 раза по сравнению с PCIe 4.0.

×

🤖 Гиперреалистичный AI

Поколение RTX 50xx получит улучшенные Tensor ядра для нейросетей, улучшение DLSS 4. AMD добавит FSR 4 на AI ускорителях.

×

💡 NVIDIA Blackwell (2024-2025)

Новая архитектура с техпроцессом 3 нм (TSMC), до 200 млрд транзисторов, чиплетный дизайн. Планируется выход RTX 5090 с 192-256 ядрами SM.

×

💡 AMD RDNA 4 / UDNA

RDNA 4 (2024-2025) — улучшенная производительность лучей, переход на UDNA — унификация геймерских и AI архитектур.

×

💡 Intel Battlemage / Celestial

Battlemage (2025) — улучшенная производительность, Celestial (2026) — флагманская серия. Планируется конкуренция с RTX 50xx.
